# ADR-0003: 적자 매매 방지 가드 — 강한 신호 매도 / 리밸런싱 / 트레일링 스탑 floor

| 항목 | 값 |

| 날짜 | 2026-01-25 |
| 도메인 | trading (코인) |
| 관심사 | 알고리즘 / 리스크 |

## Context — 무엇이 문제였나

운영 중 누적된 *손실 패턴* 분석 결과 세 가지 시나리오에서 봇이 적자를 확정 짓고
있었다.

1. **강한 매도 신호 + 손실 상태** — 점수상 매도 강신호이지만 진입가 대비 −1% 인 시점에
   매도 → 슬리피지·수수료 합산 −1.5% 손실 확정.
2. **리밸런싱 매도 + 손실 포지션** — 비중 조정 목적 매도가 손실 포지션을 처분하면서
   평균 진입가 끌어올리지 않음 → 단순 손실 청산.
3. **트레일링 스탑이 손익분기점 아래** — 진입 직후 수수료 반영 못해 트레일링 스탑이
   진입가 + 왕복수수료 *아래로* 내려감 → 작은 변동에도 자동 청산되며 적자.

## Decision — 무엇을 골랐나

세 시나리오 각각에 가드 추가.

- **강한 신호 매도 가드:** 매도 시 평가손익 ≥ −2% 일 때만 실행. (−2% 미만 손실 포지션
  은 *의도된 손절* 만 트리거 — 강신호로 우발 매도 X.)
- **슬리피지 반영:** 매수/매도 판단 시 0.5% 슬리피지 반영해 *예상 체결가* 기반 결정.
- **리밸런싱 가드:** 리밸런싱 매도 시 포지션 평균 손익률 ≥ 0% 일 때만 실행.
- **트레일링 스탑 floor:** 트레일링 스탑이 *진입가 + 왕복수수료(0.5%)* 아래로 내려가지
  않도록 보장.

## Rationale — 왜 이 선택인가

| 대안 | 장단점 | 기각 이유 |
|---|---|---|
| 신호 임계만 강화 | 단순 | 신호 자체는 정확 — 문제는 손실 시점 매도 |
| 손절(stop-loss) 비활성 | 작은 손실 회피 | 큰 손실 노출 |
| **(선택) 시나리오별 가드 3종** | 정확한 트레이드오프 | — |

각 가드는 *손실 회피* 와 *기회 비용* 사이의 트레이드오프. -2% 임계는 백테스트 결과
누적 손실 가장 큰 영향 구간.

## Consequences — 영향

- **긍정:**
  - 누적 손실 감소.
  - 트레일링 스탑이 손익분기점 아래로 내려가지 않음 → 작은 변동에도 우발 청산 X.
- **부정:**
  - 강한 매도 신호 적시 대응 못 하는 경우 발생 — 손실 -2% 이하 포지션이 -5% 까지 갈
    위험.
  - 임계 0.5% / -2% 는 KRW-ADA 기준 — 다른 페어 적용 시 재튜닝.
- **후속:**
  - ADR-0001 (8지표 컨센서스) 의 신호 정확도와 이 ADR 의 가드는 *독립* — 같이 작동.
