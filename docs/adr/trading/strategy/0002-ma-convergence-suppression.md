# ADR-0002: MA 수렴 시 (|MA5−MA20|/MA20 < 0.2%) 크로스 점수 0 으로 억제

## Context — 무엇이 문제였나

ADR-0001 (8지표 채점) 의 MA 크로스 점수가 *횡보장에서* 매 캔들마다 작은 변동만으로도
크로스/역크로스 반복 발생 → 점수가 ±25 사이로 흔들려 매매 실행이 자주 발동.

원인: MA5 와 MA20 이 거의 같은 값일 때(횡보장) 작은 노이즈가 크로스 의미 없는 신호를
생산.

## Decision — 무엇을 골랐나

MA 수렴 임계 검출 + 크로스 점수 억제.

- **수렴 조건:** `|MA5 − MA20| / MA20 < 0.2%`.
- **수렴 시 크로스 점수 = 0** — 이벤트(+25)/상태(+10) 모두 차단.
- 다른 지표(RSI, MACD 등) 점수는 영향 없음.
- **쿨다운 로직 동시 도입** — 직전 매매 후 일정 시간(분) 안에는 같은 방향 매매 차단.

## Rationale — 왜 이 선택인가

| 대안 | 장단점 | 기각 이유 |
|---|---|---|
| MA 가중치 ↓ | 모든 시장에서 MA 영향 감소 | 추세 시장의 정상 신호도 약해짐 |
| 더 긴 MA 사용 (50/200) | 횡보장 무감각 | 짧은 추세 잡지 못함 — 단타 전략과 충돌 |
| **(선택) 수렴 검출 + 조건부 억제** | 횡보장만 억제, 추세장 그대로 | — |

0.2% 임계는 ADA 변동성에서 *MA 차이가 의미 없다* 고 판단되는 경계 (백테스트 기반).
다른 코인으로 확장 시 임계 재튜닝 필요.

## Consequences — 영향

- **긍정:**
  - 횡보장 매매 빈도 감소 → 수수료 손실 감소.
  - 쿨다운 로직과 합쳐져 whipsaw(반복 손실 매매) 가드.
- **부정:**
  - 임계 0.2% 가 코인별 다름 → KRW-ADA 외 적용 시 재튜닝.
- **후속:**
  - ADR-0003 (적자 매매 방지) 가 *매매가 일어나도 손실로 끝나지 않게* 추가 가드.
