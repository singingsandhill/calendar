목표는 이 프로젝트의 `coin` 성격을 가진 모듈, 즉 실제로는 `trading` 모듈을 분석하여

- 절대 일반적인 업비트/바이낸스 스타일 수동 매매 화면을 제안하지 마세요.

- 따라서 UI 개선 방향도 수동 주문 UX보다
  **전략 상태 가시성, 실행 안정성, 위험 제어, 결과 확인, 운영 판단 효율** 중심이어야 합니다.
