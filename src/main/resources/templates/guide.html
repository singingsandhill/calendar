DateDate 사용 가이드

5단계로 여러명의 약속을 쉽게 잡는 방법

1

페이지 만들기

datedate.site에 접속하여 원하는 ID를 입력하고 시작하기를 클릭합니다.

  • 영문 소문자, 숫자, 하이픈(-)만 사용 가능
  • 2~20자 사이로 입력
  • 예시: my-team, college-friends, trip-2025

랜덤 생성 버튼으로 고유한 ID를 자동으로 만들 수도 있습니다.

2

일정 생성

대시보드에서 원하는 월(月)을 선택하면 해당 월의 일정 페이지가 자동으로 생성됩니다.

  • 여러 달의 일정을 동시에 관리 가능
  • 대시보드에서 모든 일정을 한눈에 확인
3

링크 공유

생성된 일정 페이지의 링크를 복사하여 참여자들에게 공유합니다.

  • 카카오톡, 슬랙, 이메일 등 어디로든 공유 가능
  • 참여자는 가입이나 앱 설치 없이 바로 참여
  • 링크만 있으면 누구나 접근 가능
4

날짜 선택 & 투표

참여자들이 이름을 입력하고 캘린더에서 가능한 날짜를 클릭합니다.

  • 최대 8명까지 참여 가능
  • 참여자마다 고유 색상이 부여되어 직관적으로 확인
  • 날짜 수정도 간편 - 드롭다운에서 이름 선택 후 재선택
  • 장소와 메뉴도 제안하고 투표할 수 있음
5

결과 확인 & 결정

가장 많은 사람이 가능한 날짜가 자동으로 하이라이트됩니다.

  • 겹치는 날짜가 많을수록 진한 색상으로 표시
  • 인기 장소와 메뉴 순위도 함께 확인
  • 최종 약속 날짜, 장소, 메뉴를 결정하세요
